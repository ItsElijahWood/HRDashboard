# HR Dashboard Backend

## Overview

The HR Dashboard is a web-based application designed to streamline employee management. This backend serves as the core of the HR Dashboard, providing secure user authentication and management of employee data. The system allows authorized users to sign in, edit, and manage employee records in a secure and efficient manner.

## Upcoming Features

- **Sign-In**: Secure login for authorized users to access the dashboard.
- **Employee Information Management**: Edit, update, and manage employee details such as personal information, position, department, and contact details.
- **Security**: Protects sensitive employee information using secure authentication methods.

## Tech Stack

- **Node.js**: JavaScript runtime for building the backend.
- **Express.js**: Web application framework for Node.js to handle routes and HTTP requests.
- **MongoDB**: NoSQL database to store employee records.
- **JWT (JSON Web Tokens)**: Secure authentication method for protecting endpoints.
- **bcrypt**: Password hashing library for secure user authentication.
